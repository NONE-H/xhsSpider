JavaScript是前端开发的一门脚本语言，遵循欧洲计算机供应商协会（ECMA）发布的ECMA-262规范，是当下使用最广泛的前端脚本语言。该语言隶属于C系编程语言，有许多语法格式与C语言类似。学习前端开发的过程中，CSS以后的所有内容几乎都是对该门语言的讲解。正确的JavaScript语言的学习路线如下。【第一阶段 JavaScript语法基础】该阶段主要学习有关JS脚本语言的基础知识，认识数据类型和常变量，学习JS的运算体系，学习结构化程序设计原理（顺序结构、选择结构、循环结构），学习基础算法，认识数组数据及其操作方法，认识对象数据及其操作方法，学习模块化程序设计原理（函数），掌握正则表达式在JavaScript脚本语言中的使用规范。【第二阶段 DOM文档对象模型】该阶段主要学习使用JS操纵DOM节点的相关知识，认识节点的类型，熟练使用三大节点（元素节点、属性节点、文本节点）的属性和操作方法，熟练操作表格，熟练操作表单，能够通过JS操作CSS样式，掌握BOM（浏览器对象模型）中各对象的属性和操作方法。【第三阶段 基于对象的程序设计】该阶段将涉及到JavaScript的本质特性和核心概念，是较为高级的JavaScript知识。需要学员学习内置类的属性和操作方法，使用自定义对象编程，使用自定义类编程，对基本开发模式有所了解（构造函数模式、原型模式、混合模式、简单工厂模式、单例模式），掌握变量的作用域与生存期，理解变量的本质与闭包，深度理解属性的特性（数据特性、访问器特性），熟练使用Object类，掌握类的继承。【第四阶段 HTML5技术】该阶段主要学习JavaScript有关H5技术的新知识。包括音视频操作，Canvas位图绘制，Storage本地存储，File文件操作，Drag拖拽模型等。【第四阶段 ES6新规范】该阶段主要学习ES6（ES 2015）对脚本语言提出的新规范，包括各种数据类型的扩展，解构操作，使用Class创建类，Set和Map数据结构及应用，可遍历对象Iterator，Promise技术，Proxy与属性的特性，Module模块标准与加载实现等。JavaScript的学习是个漫长的过程，涉及到的知识点和原理多而繁杂，学习过程中要多总结、多思考，只有学好了JavaScript，才能在前端开发的道路上有质的飞跃。

我是一个有十年开经验的老程序猿，在公司待过很多实习生、应届生，也兼职在当培训老师带学生，以下内容分享给准备学习 Vue 的童鞋。1、如果 JavaScript 还比较不熟练的话，先花两周的时间把 JavaScript 再提升一下，建议直接看 《JavaScript高级程序设计》；2、看 Vue 的官方文档，Vue的官方文档对我们中国人很友好，比 React 的文档要更友好一些，基本上接近大白话了。直接新建一个 HTML，然后引入 Vue，开始学习基础用法。这里强调一下：一开始学习的时候不要使用 vue-cli 创建，尤其是在你还不熟悉基于 Node.js 的构建工具时，不要给自己增加复杂度，直接在 script 标签中引入就好了。3、学习基础语法的时候重点把双向绑定、data、props、条件（v-if、v-else）、循环（v-for）、事件处理、计算属性、watch先学一遍，学完这些之后，对 Vue 算是有一些了解了，能写出最基本的代码了。4、有了以上的基础之后，就可以开始学习组件、插槽等内容，这里主要是要了解一下组件化的思想。5、从我带学生的经验来看，学完上面的内容，应该要两周左右的时间，接着就可以使用 vue-cli 去学习了，这个过程中需要了解一下 webpack，至少知道 loader、plugin 是什么，对 .vue 文件是编译成 js 文件要有最基本的了解。6、开始做小项目，我一般会建议先做个 todo-list，做这个小项目基本上可以把学到的所有基础知识用上去。7、学习一下 axios 是怎么用的，公司里肯定会对 axios 进行一层封装，不过不难。8、这时候可以进行业务开发了，写一些基础的业务，简单的组件应该不成问题了。写三个月左右，可以稍微熟练一些了。9、你大概率情况下会用开源的组件库，比如 Element UI、IView、Ant Design Vue，你有一些经验了，应该去尝试看看组件库的源码，特别是当你遇到问题的时候，带着问题去看组件库的源码，你应该去学习一下这个组件是怎么实现的，对组件源码熟悉了，对你后续遇到问题的时候会更有思路。学习高手是怎么封装组件的，对你的业务开发有指导作用。#前端 #web前端学习 #自学前端
